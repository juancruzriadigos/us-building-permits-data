# US Building Permits & Local Fees Data 🏗️

Welcome to the public documentation overview for **[StateDataIndex.com](https://statedataindex.com)**. 

StateDataIndex is a comprehensive programmatic directory that helps homeowners and contractors calculate municipal permit fees for various home improvement projects across the United States.

## 📌 Main Features
* **Permit Cost Calculators:** Dynamically estimate fees for projects like [Roof Replacements](https://statedataindex.com), [Solar Panel Installations](https://statedataindex.com), and HVAC upgrades based on local county multipliers.
* **Zoning Laws & Building Codes:** Detailed breakdowns of local climate risks (Hurricane, Snow, Earthquake) and their impact on construction regulations.
* **National Coverage:** Processing data spanning across major states including Texas, Florida, and California.

## 🛠️ Data Structure Overview
Our database cross-references core tables to provide accurate local data:
1. `States` (Base fee multipliers, state sales tax, climate risks)
2. `Counties` (Population density, local agricultural exceptions)
3. `Projects` (Base costs, permit probability)
